Локализация текстов программы

1. Сделайте резервную копию файла Strings.template.txt в другую папку
2. Переименуйте Strings.template.txt в зависимости от языка (список языков и аббривиатур в файле culture.txt)
    пример: Strings.uk-UA.txt
3. Запустите run.bat с именем Вашего файла в качестве параметра
    пример: run.bat Strings.uk-UA

ВНИМАНИЕ: в п.3 разрешение файла(.txt) указывать НЕ НУЖНО!
