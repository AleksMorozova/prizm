Локализация текстов программы

1. Сделайте резервную копию файла Strings.template.txt

2. Переименуйте Strings.template.txt в зависимости от языка (список языков и аббревиатур находится в файле cultures.txt)
    пример: Strings.uk-UA.txt

3. Отредактируйте файл, добавив в него переводы всех необходимых строк в виде текста после знака =

4. Запустите run.bat с именем Вашего файла в качестве параметра
    пример: run.bat Strings.uk-UA

ВНИМАНИЕ: в п.3 расширение файла(.txt) указывать не нужно.
