Локализация текстов программы

Локализация программы проводится в два этапа. 
	Первый - локализация внутренних сообщений и надписей в программе. 
	Второй - локализация базовых элементов управления программой.

ВНИМАНИЕ: на некоторых операционных системах, в зависимости от настроек, могут потребоваться права администратора системы. 

Базовое значение языка программы - русский и в отдельном файле перевода не нуждается. Локализация элементов управления на русский поставляется вместе с программой.

I. Локализация текстов

	1. Для создания шаблона перевода Запустите программу с ключом 'template' [Prism.Program.exe template] без квадратных скобок.
	В папке Languages появится файл Strings.template.txt - это заготовка файла перевода.

	2. Сделайте резервную копию файла Strings.template.txt

	3. Переименуйте Strings.template.txt в зависимости от языка будущего перевода. Список языков и аббревиатур находится в файле cultures.txt папка Languages. Правильное написание указанно в первом столбце (Language)
		пример: Strings.uk-UA.txt

	4. Отредактируйте файл, добавив в него переводы всех необходимых строк в виде текста после знака =

	5. Запустите run.bat из папки Languages с именем Вашего файла в качестве параметра
		пример: [run.bat Strings.uk-UA] без квадратных скобок и расширения файла (.txt)
		
	ВНИМАНИЕ: в п.3 расширение файла(.txt) указывать не нужно.

II. Локализация элементов управления

	1. Авторизуйтесь на площадке http://localization.devexpress.com/, при необходимости зарегистрируйтесь

	2. Подтвердите лицензионное соглашение

	3. Нажмите 'Add translation' 

	4. В поле 'Version' выберите '2014.2'. В поле 'Culture' требуемый язык.

	5. Выберите созданную версию перевода и нажмите 'Download'

	6. Файлы локализации поступят на почтовый ящик указанный при регистрации на  http://localization.devexpress.com/

	7. Распакуйте полученную локализацию в папку с установленной программой по аналогии со стандартными языковыми пакетами
		пример: папка ru в корне программы
