Процесс компиляции:
1)	Необходимо установить гитхаб клиент и клонировать два репозитория – prizm и prizm_external.
2)	Запускаем PrizmMainSolution из Visual Studio. Переключаемся в режим Release и перестраиваем решение (Build – Rebuild solution).
В случае если какие-то библиотеки не подключены, необходимо использовать консоль менеджера пакетов (Tools – Library Package Manager – Package Manager Console) для подключения недостающих библиотек. Библиотеки подключаются следующим образом: 
PM> Install-Package Название _ Библиотеки -Version Номер _ Версии
После установки недостающих библиотек, перестраиваем решение еще раз. 
3)	После успешной перестройки, переходим в папку install и запускаем "!CreateCompileScript.cmd" скрипт. 
Если необходимо, в этом скприпте можно изменить путь к папке prizm_external и к библиотекам devExpress 
4)	Запускаем созданный скрипт «!Compile. cmd»
5)	Запускаем файл установки из созданной папки release.

Клонирование репозитория:
1) создаем папку в которую поместим клонированный репозиторий
2) в GitHub нажимаем на плюсик в левом верхнем углу и выбираем «Clone», в списке находим нужный репозиторий и клонируем его указав созданную для него папку.  
   в Git Extensions выбираем «Clone repository». В строке «Repository to clone» указываем адрес репозитория, в «Destination» - папку созданную для репозитория, а в «Branch» - ветку из репозитория, которую мы хотим клонировать
